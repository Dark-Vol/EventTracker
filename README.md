# Облік подій (Event Tracker)

Веб-додаток для обліку подій з відображенням статистики. Реалізовано на Angular 21 з використанням localStorage для збереження даних.

## Функціонал

- ✅ Форма додавання подій з полями: назва та тип (click, lead, sale)
- ✅ Таблиця відображення всіх подій з датою та часом створення
- ✅ Блок статистики з загальною кількістю подій та кількістю по кожному типу
- ✅ Збереження даних у localStorage
- ✅ Можливість видалення подій

## Як запустити проект локально

### Вимоги

- Node.js (версія 18 або новіша)
- npm (версія 9 або новіша)

### Крок 1: Клонування репозиторію

Якщо ви клонуєте репозиторій:

```bash
git clone <repository-url>
cd testing-pro
```

### Крок 2: Встановлення залежностей

```bash
npm install
```

### Крок 3: Запуск локального сервера розробки

```bash
npm start
```

або

```bash
ng serve
```

### Крок 4: Відкриття в браузері

Після запуску сервера відкрийте браузер і перейдіть за адресою:

```
http://localhost:4200/
```

Додаток автоматично перезавантажується при зміні файлів вихідного коду.

## Збірка проекту

Для створення production збірки:

```bash
npm run build
```

або

```bash
ng build
```

Зібрані файли будуть збережені в директорії `dist/`. Production збірка оптимізована для продуктивності та швидкості.

## Деплой на Vercel

### Варіант 1: Деплой через Vercel CLI

1. Встановіть Vercel CLI:
```bash
npm i -g vercel
```

2. Увійдіть в обліковий запис:
```bash
vercel login
```

3. Задеплойте проект:
```bash
vercel
```

4. Для production деплою:
```bash
vercel --prod
```

### Варіант 2: Деплой через GitHub

1. Завантажте код на GitHub
2. Перейдіть на [Vercel](https://vercel.com)
3. Натисніть "New Project"
4. Імпортуйте ваш GitHub репозиторій
5. Vercel автоматично визначить Angular проект і налаштує збірку
6. Натисніть "Deploy"

### Налаштування збірки для Vercel

Vercel автоматично визначить Angular проект. Можна створити файл `vercel.json` для додаткових налаштувань:

```json
{
  "buildCommand": "npm run build",
  "outputDirectory": "dist/testing-pro/browser",
  "framework": "angular"
}
```

## Технології

- **Angular** 21.1.0
- **TypeScript** 5.9.2
- **SCSS** для стилізації
- **localStorage** для збереження даних

## Структура проекту

```
src/
├── app/
│   ├── events/

│   │   ├── events.component.html    # Шаблон компонента
│   │   └── events.component.scss    # Стилі компонента
│   ├── app.ts                       # Головний компонент
│   ├── app.routes.ts                # Маршрутизація
│   └── app.html                     # Шаблон головного компонента
└── styles.scss                      # Глобальні стилі
```
